This portfolio demonstrates the use of SQL views, stored procedures, and stored functions to manage and manipulate database data. It covers tasks such as filtering data with views, updating records with stored procedures, and retrieving data using functions.

Step-by-Step Process:

  1. Setup MySQL Workbench:
    • Start MySQL server from XAMPP.
    • Connect MySQL Workbench to the server.
    • Execute practice queries using democodes.sql.
  2. Use Inventory Database:
    • Open inventory.sql to begin the tasks.
  3. Create Views:
    • Vendor/product data from 2002 onwards
    • Products with prices between 100–150
    • Total price of products by vendor code
  4. Create Stored Procedure: Update vendor name
  5. Create Function: Display product info based on parameters
  6. Execute and Document: Capture screenshots of results
