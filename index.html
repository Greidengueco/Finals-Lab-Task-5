Finals Lab Task 5

This portfolio demonstrates the use of SQL views, stored procedures, and stored functions to manage and manipulate database data.

Step-by-Step Process

1. Setup MySQL Workbench

  • Start MySQL server using XAMPP
  • Open Workbench and connect
  • Run democodes.sql

2. Use Inventory Database

3. Create Views

  • Vendor and product info (2002 onwards)
  • Products priced 100–150
  • Total price = P_ONHAND x P_PRICE (specific vendors)

4. Create Stored Procedure

  • Update vendor name 'Bryson, Inc.' to 'Bryson and Co'

5. Create Function

  • Function takes vendor code + state → shows product details
