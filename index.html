  • 2. Use the inventory.sql database for the tasks.

Task 2: Create a view for products with prices between 100-150

Task 3: View for total price of products sold by vendors (v_code: 21344, 23119, 24288)

Task 4: Stored Procedure to update vendor name

Task 5: Function using vendor code and state to display product details
